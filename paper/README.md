# paper/

- **`joss/`** — the actual JOSS submission: `paper.md` + `paper.bib`, compiled
  `paper.pdf` and `jats/paper.jats`. This is the only paper submitted anywhere.
- **`drafts/`** — earlier, unsubmitted paper drafts (`paper.md`, `paper.tex`,
  `references.bib`) and the venue-selection memo (`VENUE_MEMO.md`) written
  before JOSS was chosen as the venue. Kept for history, not maintained.
- **`research/`** — working notes used to research and justify each method
  added to the library:
  - `EVIDENCE_TABLE.md` — per-claim source tracing.
  - `surveys/` — per-method literature surveys (`NEW_METHOD_SURVEY*.md`).
  - `implementation_prompts/` — the implementation prompts used to add each
    method (`IMPLEMENTATION_PROMPT_*.md`).
